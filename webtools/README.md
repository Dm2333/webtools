# webtools

实现的功能
=============
		1.从搜索引擎中提取网址
		2.查询一个网站的同站
1.搜索引擎中提取网址
--------------
		实现了从百度和好搜的搜索结果中提取网址
2.查询一个网站的同站
----------------
		实现了从爱站提取一个网址的同站，并且加了自动验证网址能否链接的功能。
		如果不能访问就是红色的。
